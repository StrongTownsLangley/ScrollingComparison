can you make a python script that will take an image called "corridor_vert_1080_before.png" and scroll it from the top of the image to the bottom and output a 60fps video? the output resolution of the video should be 1080p portrait mode (1080x1920) and the image size is 1080x15324

Sure, here's a Python script that uses OpenCV to create a 60fps video from a scrolling image. The output resolution of the video will be 1080x1920 (portrait mode), and the image "corridor_vert_1080_before.png" will be scrolled from top to bottom.

python
import cv2
import numpy as np

# Load the image
image_path = 'corridor_vert_1080_before.png'
image = cv2.imread(image_path)

# Parameters
fps = 60
frame_width = 1080
frame_height = 1920
output_video = 'scrolling_video.mp4'

# Get image dimensions
img_height, img_width, _ = image.shape

# Calculate the number of frames
total_frames = img_height - frame_height + 1

# Define the codec and create VideoWriter object
fourcc = cv2.VideoWriter_fourcc(*'mp4v')  # You can use other codecs as well
video_writer = cv2.VideoWriter(output_video, fourcc, fps, (frame_width, frame_height))

for i in range(total_frames):
    # Extract the frame from the image
    frame = image[i:i + frame_height, 0:frame_width]
    
    # Write the frame to the video
    video_writer.write(frame)

# Release the video writer object
video_writer.release()

print(f"Video saved as {output_video}")
